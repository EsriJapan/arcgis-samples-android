## HeatmapSample

このサンプルは、ヒートマップの配色をカスタマイズして、任意のフィーチャ レイヤーにレンダリングして表現するサンプルです。
レンダリング要素であるJSONは、そのままお使いいただけます。

### 使用する主なクラス
 * FeatureLayer
 * FeatureTable
 * Renderer
 * MapView
 * ArcGISMap

### 使用するフィーチャレイヤー
 * https://services.arcgis.com/wlVTGRSYTzAbjjiC/arcgis/rest/services/tokyohoikuen23/FeatureServer/0
  >※ArcGIS Open Data(http://hub.arcgis.com/pages/open-data)の東京23区の保育園(http://hub.arcgis.com/datasets/ejopendataportal::%E4%BF%9D%E8%82%B2%E5%9C%9223%E5%8C%BA?geometry=138.6%2C35.482%2C140.586%2C35.872)より

### 設定
 特に不要
